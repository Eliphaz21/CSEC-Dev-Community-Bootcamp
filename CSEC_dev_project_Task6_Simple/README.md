# Simple Express.js MVC Backend

This project is a simple backend server built using **Express.js** to understand
basic backend concepts, routing, and the **MVC (Model–View–Controller) pattern**.

---

## What is MVC (Very Simple)

MVC means separating responsibilities:

- **Routes** → decide where the request goes
- **Controllers** → decide what response to send
- **Models** → handle data (not used yet)

This project uses only **Routes + Controllers** (beginner level).
