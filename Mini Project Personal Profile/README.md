# Personal Profile Mini Project

This is a simple personal profile webpage built using HTML, CSS, and JavaScript.
It was created as a mini project to practice basic web development concepts.

## Technologies Used

- HTML
- CSS3
- JavaScript

## Setup Instructions

1. Clone or download the project
2. Place your profile image inside the `assets` folder and name it `profile.jpg`
3. Open `index.html` in your browser
